# Cucumber-JS

Cucumber-JS is available as an [npm package](https://www.npmjs.com/package/@cucumber/cucumber) for use in Node.js.

Add `@cucumber/cucumber` as a development dependency:

```shell npm2yarn
npm install --save-dev @cucumber/cucumber
```
